# Sicfler backend

## Getting Start

```shell
npm i --save
```

### How to

### Launch on local

```shell
sls offline
```

### Deploy

```shell
sls deploy
```

### Type Generate

- Server

```shell
npm run gen-types:server
```

- Client

```shell
npm run gen-types:client
```

## Tech stacks

- TypeScript
- ApolloServer
- Graphql-codegen
- Serverless framework
